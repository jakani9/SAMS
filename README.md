# 学習塾 入退室管理アプリ

QRコードをPCのWebカメラで読み取り、来塾・帰宅を自動判定して記録するスタンドアロンアプリです。

## 機能

- **QRスキャン** — 生徒ID入りのQRコードをWebカメラで読み取り
- **自動判定** — 同じ日の1回目＝来塾、2回目＝帰宅（3回目以降も交互に判定）
- **記録** — 日付・時刻・生徒ID・氏名・区分を保存（ブラウザ内）
- **生徒管理** — 生徒IDの登録と本日の滞在状況（在塾中・帰宅済み・未来塾）
- **CSV出力** — ダウンロード、またはファイルへの自動追記（Googleスプレッドシートへインポート可能）

## 使い方

### 1. 起動

**推奨:** `起動.bat` をダブルクリック

- Python がインストールされていれば、自動的にローカルサーバーを起動してブラウザが開きます
- Python がない場合は `index.html` を直接開きます

**手動:** ターミナルで以下を実行してからブラウザで `http://localhost:8765` を開く

```bash
cd juku-attendance
python -m http.server 8765
```

> **注意:** セキュリティ上、多くのブラウザではカメラは `http://localhost` または HTTPS でのみ利用できます。`file://` で直接開くとカメラが使えない場合があります。

### 2. 生徒を登録・滞在状況を確認

1. 「生徒管理」タブを開く
2. 下部の「生徒の登録」で生徒ID（例: `S001`）と氏名を入力して「追加」
3. 上部の一覧で本日の滞在状況を確認（在塾中 / 帰宅済み / 未来塾）

滞在状況は本日の最終スキャンから自動判定されます。来塾スキャン後は「在塾中」、帰宅スキャン後は「帰宅済み」、本日未スキャンは「未来塾」です。

QRコードの中身は **生徒IDのみ** です。QRコードは別途作成して生徒に配布してください。氏名は生徒マスタのみに保存し、CSVには含まれません。

### 3. スキャン

1. 「スキャン」タブで「カメラ開始」をクリック
2. カメラの使用を許可
3. 生徒にQRコードをかざしてもらう
4. 画面に「来塾」または「帰宅」が表示され、記録されます

同じQRを連続で読み取らないよう、3秒間のクールダウンがあります。

### 4. 記録の保存（Googleスプレッドシート）

#### 方法A: CSVダウンロード

「記録・CSV」タブ → 「CSVダウンロード」→ Googleスプレッドシートで「ファイル → インポート」

#### 方法B: CSV自動保存（推奨・Chrome / Edge）

「CSV自動保存を設定」→ 保存先CSVファイルを指定 → 以降、スキャンのたびに自動追記

Googleドライブ上のCSVを指定し、スプレッドシートでそのCSVを参照することもできます。

## CSV形式

```csv
日付,時刻,生徒ID,区分
2026-06-30,14:30:00,S001,来塾
2026-06-30,17:00:00,S001,帰宅
```

CSVは UTF-8（BOM付き）で保存されます。Excel で文字化けする場合は、一度 CSV自動保存を再設定するか、Excel の「データ → テキストファイル」から UTF-8 を指定して取り込んでください。

## 必要な環境

- **OS:** Windows 10 / 11（Mac でも `index.html` または Python サーバーで動作）
- **ブラウザ:** Google Chrome または Microsoft Edge（推奨）
- **カメラ:** PCに内蔵または外付けWebカメラ
- **オプション:** Python 3（`起動.bat` 用。なくても HTML 直接起動は可能）
- **インターネット:** 不要（QR 読み取りライブラリは同梱）

## データの保存場所

記録と生徒情報は **ブラウザの localStorage** に保存されます。ブラウザのデータを削除すると記録も消えます。重要な記録は定期的にCSVでバックアップしてください。

## 費用

完全無料で利用できます。外部 CDN への接続は不要です。
